## Question

[題目連結](https://leetcode.com/problems/roman-to-integer/)

寫出羅馬數字，規則如下：

| Symbol | Value |
| ------ | ----- |
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |

```
Input: s = "III"
Output: 3
Explanation: III = 3.
```

```
Input: s = "LVIII"
Output: 58
Explanation: L = 50, V= 5, III = 3.
```

## Answer

```js
var romanToInt = function (s) {
  const map = {
    I: 1,
    V: 5,
    X: 10,
    L: 50,
    C: 100,
    D: 500,
    M: 1000,
  };
  var res = 0;
  const splitArr = s.split("");
  splitArr.forEach((item, index, arr) => {
    // ["L", "V", "I", "I", "I"]
    if (!arr[index + 1]) return (res += map[item]);
    if (map[item] < map[arr[index + 1]]) {
      res -= map[item];
    } else {
      res += map[item];
    }
  });
  return res;
};
```
